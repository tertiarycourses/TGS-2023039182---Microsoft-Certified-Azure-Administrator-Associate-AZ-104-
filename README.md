# Microsoft Certified: Azure Administrator Associate (AZ-104) Hands-On Labs Summary

## Course Overview
This resource provides **26 step-by-step lab exercises** for the WSQ Microsoft Certified Azure Administrator Associate (AZ-104) Training course (TGS-2023039182) delivered by Tertiary Infotech Academy. All labs run in the **Azure Portal** together with **Azure Cloud Shell** (Azure CLI and PowerShell), requiring no local installation. An [Azure free account](https://azure.microsoft.com/free/) or an instructor-provided subscription is all you need.

## Lab Structure
The curriculum spans the five official AZ-104 exam skill areas (skills measured as of April 17, 2026):

1. **Manage Azure identities and governance (20–25%)** — 6 labs covering Microsoft Entra users/groups, external users, SSPR, RBAC, Azure Policy, subscriptions, resource groups, locks, tags, cost management, and management groups
2. **Implement and manage storage (15–20%)** — 5 labs on storage accounts, redundancy, storage firewalls, SAS tokens, access keys, Blob containers, tiers, lifecycle, versioning, Azure Files, and AzCopy / Storage Explorer
3. **Deploy and manage Azure compute resources (20–25%)** — 6 labs on ARM templates and Bicep, virtual machines, disks, availability zones/sets, Virtual Machine Scale Sets, containers (ACR/ACI/Container Apps), and App Service
4. **Implement and manage virtual networking (15–20%)** — 5 labs on virtual networks, subnets, peering, public IPs, user-defined routes, NSGs, ASGs, Bastion, service/private endpoints, Azure DNS, and load balancing
5. **Monitor and maintain Azure resources (10–15%)** — 4 labs on Azure Monitor metrics/logs/alerts, Network Watcher, Azure Backup with Recovery Services vaults, and Azure Site Recovery

## Key Features
- Every lab is completed with the **Azure Portal**, **Azure CLI**, and **Azure PowerShell** so learners practise all three interfaces the exam expects
- Commands are copy-paste ready in **Azure Cloud Shell** — no SDK installs on your own machine
- Each lab maps directly to a published AZ-104 *Skills measured* objective and ends with clean-up steps to avoid unnecessary charges
- Aligned with the official Microsoft Learning content: https://github.com/MicrosoftLearning/AZ-104-MicrosoftAzureAdministrator
- Registration available at: https://www.tertiarycourses.com.sg/wsq-microsoft-azure-administrator-training.html

## Important Note
Always delete or deallocate resources when a lab is finished — Azure bills for running VMs, public IPs, gateways, and provisioned storage. Perform every lab only in a subscription you own or one your instructor has authorized. The clean-up section at the end of each lab removes the resource group created for that exercise.
